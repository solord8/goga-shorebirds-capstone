# GOGA shorebirds capstone
The goal of this project is to demonstrate analytical proficiencies using a large ecological dataset while contributing to shorebird management at Golden Gate National Recreation Area (GOGA).

Important files: Datasets used for this project are publicly available at the NPS Datastore   (linked here: https://irma.nps.gov/DataStore/Reference/Profile/2315177)

To use this workflow: Clone repository and use the numbered .R scripts sequencially. Set the goga-shorebirds-capstone folder as the working directory to ensure file paths work as intended.
